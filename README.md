# Yedikare Remote

Yedikare müşterilerine teknik destek vermek için kullanılan markalı uzak masaüstü istemcisi.

Bu proje, [HopToDesk](https://www.hoptodesk.com) v1.46.1 kaynak kodunun yeniden markalanmış bir çatallamasıdır (fork). Bağlantılar HopToDesk'in genel relay ağı üzerinden kurulur; sunucu altyapısında değişiklik yapılmamıştır.

## Upstream'e göre değişiklikler

- Uygulama adı, ikonlar ve tema rengi Yedikare markasına uyarlandı
- Uygulama içi website/gizlilik linkleri [yedikare.com.tr](https://www.yedikare.com.tr) adresine yönlendirildi
- Çeviri metinlerindeki uygulama adı çalışma anında değiştirilir (`src/lang.rs`), dil dosyaları upstream ile uyumlu kalır
